Chèn nhạc vào bài post | Plugin hexo-tag-mplayer 🎧

|Số từ:515|Thời gian đọc:3min|Số người xem:


Như tiêu đề thì plugin này giúp chèn cPlayer vào trang Hexo bằng cách sử dụng tags.

Install

Sử dụng lệnh bash này vào PowerShell, Cmd hoặc Terminal của bạn.

  • Chắc chắn rằng bạn đã cài đặt Nodejs và npm trước đó rồi nhé!

Quick Start

Sau khi tải Plugin, các bạn vào bài post/page và chèn tags này vào bài markdown của bạn.

Có thể xoá một số thông tin nếu muốn như: lyric, sublyric, album,..

Các thông tin bắt buộc phải có là: name, artist, src và poster
Nếu bạn không muốn bị lỗi vui lòng đừng xoá!

Mucsics:

Videos

Để chèn video thì các bạn thay thế như sau:

  • Đường dẫn đến file MP3 các bạn thay bằng MP4

Nếu muốn xem Demo, vào trang KawaiiCassie Musics này.

Option

Option Default content Description
playlist [] Playlist.
autoplay false Tự động phát (không hỗ trợ trên moblie).
volume 1 Âm lượng mặc định.
playmode listloop Chế độ phát mặc định (có 3 chế độ: listloop, singlecycle, listrandom).
big false Kích thước mặc định (muốn to hơn thì chỉnh true).
dark false Màu sắc của player (chế độ tối thì true).
  • 3 Chế độ mode:
    • listloop Vòng lặp cho playlist
    • singlecycle Vòng lặp cho một bài hát
    • listrandom Phát ngẫu nhiên

Ngoài ra còn nhiều Option nữa. Các bạn vào đây tham khảo CPlayer Option
Để xem thêm chi tiết, vui lòng vào CPlayer Demo của tác giả để xem.

Cám ơn các bạn đã đọc. Có thắc mắc gì vui lòng Comment xuống bên dưới bài viết nhé!
